create collective





name field


mandatory

alphanumeric

min 1 char max x char


collective name must be unique within the netwok, and preferably provide some insight into collective's interests

description field


mandatory

alphanumeric

min 1 char max x char


collective description might provide relevant information as to collective's focus and activities, such as describing collective's goals, information regarding membership enrollments, etc


policy majority


mandatory

integer number

ranging from 0 to 100 (default)


the number chosen will define majority minimum to be accepted as consensus for all instructions perfomed on behalf of collective


for instance, should policy majority be set 50, an instruction on behalf of the collective will only be processed after at least 50% of the collective members have voted for an acceptance of the instruction


policy super majority


mandatory

integer number

ranging from 0 to 100 (default)


the number chosen will define majority minimum to be accepted as consensus for an instruction that changes any of the collective's main information - name, description, majority and/or supermajority


reasons field


optional

alphanumeric

max x char


instruction author may provide some information as to the reasons for the instruction being proposed
