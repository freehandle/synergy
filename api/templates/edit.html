authorship field - coauthoring


option of mandatory field

selection


edit authorship might be of a group of network members


in that case author of edit instruction must provide handles for coauthors of edit and consensus policies apply for any of the further instructions related to edit


consensus policies will be performed according to the set of instruction author plus coauthors appointed


for edit instruction to be approved all appointed co-authors will have to vote for approval


authorship field - collective


option of mandatory field

selection


edit authorship might be of a collective


in that case author of edit instruction must provide the name for the collective it is signing the edit on behalf of, and collective consensus policies apply for both the approval and any of the further instructions related to draft


consensus policies will be performed according to the majority policy of the appointed collective
